COURSE DESCRIPTION
This course introduces fundamental concepts in computer science including problem-solving, algorithms, data structures, and programming. Students will learn to write programs, understand computational thinking, and apply CS principles to real-world problems.

COURSE OBJECTIVES
By the end of this course, students will be able to:
- Understand fundamental programming concepts and paradigms
- Write well-structured programs in Python
- Analyze and implement basic data structures
- Solve computational problems using algorithms
- Debug and test code effectively
- Apply computer science principles to practical scenarios

PREREQUISITES
High school algebra or equivalent. No prior programming experience required.

TEXTBOOKS AND MATERIALS
- Required: "Introduction to Computer Science" by John Guttag (3rd Edition)
- Required: Python 3.9+ (Free, open-source)
- Recommended: "Cracking the Coding Interview" by Gayle Laakmann McDowell
